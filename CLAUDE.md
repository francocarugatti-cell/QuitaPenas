# CLAUDE.md

Este archivo guía a Claude Code al trabajar en este proyecto.

## Proyecto

Quita penas 3era chance — proyecto de creación y diseño de páginas web.

## Convenciones

- Idioma de comunicación: español.
- Código y comentarios: claros y legibles.

## Stack y enfoque

- Páginas web modernas, responsive y accesibles.
- HTML5 semántico, CSS limpio (variables, flexbox/grid) y JavaScript cuando aporte valor.
- Diseño mobile-first; cuidar contraste, tipografía y jerarquía visual.

## Agentes disponibles

- `disenador-web` — Creador y diseñador de páginas web. Úsalo para maquetar,
  diseñar interfaces, proponer estructura visual y generar el código de las páginas.

## Comandos útiles

- Abrir una página en el navegador para previsualizar el resultado.
